**crosswords** is a terminal tool to help you solving crosswords.

Install
-------

.. code-block::

    pip install crosswords

To upgrade a previous installation, use:

.. code-block::

    pip install -U crosswords

Run
---

.. code-block::

    crosswords <word>

Tests
-----

Clone this repo, then: ::

    [sudo] make deps
    make check
